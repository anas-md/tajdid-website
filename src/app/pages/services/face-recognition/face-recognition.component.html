Face Recognition Attendance Sytem

This system is based on artificial intelligence (AI), which is capable of accurately detecting each person's face. The purpose of this method is to make the daily check-in and check-out process easier and less time consuming. All data will be properly saved in a centralised database once this system is in place.

As a result, companies do not need to be concerned about data loss or corruption. This system is also integrated with the dashboard system, which manages all of the data collected by this system. Furthermore, management services and human resources departments can swiftly and simply give staff attendance reports.

How does it work ?

  • The camera will remain open to detect the employee's face in real time when someone enters the camera's range.
  • The system will then record employee attendance according to time settings (office attendance policy) and send the data to the server.

Project Function

Face Detection Module

  • Detect multiple faces in real-time
  • Records the arrival and exit time settings based on the time setting

Arrival Management Module

  • Employee attendance display in real-time
  • Update attendance data as well as office entry and exit
  • Provide attendance reports
  • Manage all settings

Benefits

  • Employee attendance can be simply recorded in real-time without the need for contact.
  • Reduce the use of paper and save time throughout the check-in and check-out procedure.
  • All data should be stored in a centralised database, which is more systematic and prevents data loss.
  • Keep track of attendance at numerous selected locations.
  • The system allows you to track staff attendance in real time.
  • Quickly and simply generate staff attendance reports.
